Question 1: what are the deffarent way to select an element in DOM?

There are several ways to select elements in the DOM using JavaScript:getElementById, getElementsByClassName, getElementsByTagName, querySelector

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML sets/gets HTML markup, innerText gets visible text considering styles, and textContent gets all text without parsing HTML.

Question-3: What is event delegation in the DOM?

Attach a single event listener to a parent element to manage events from multiple child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

An event triggers on the target element and bubbles up through its parent elements. Use event.stopPropagation() to prevent this.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: const el = document.createElement('div'); Add: parent.appendChild(el); or parent.append(el); Remove: el.remove(); or parent.removeChild(el);
